Add new ESLint config for Node.js projects. It includes TypeScript support.

```bash
$ yarn add eslint @commercetools-backend/eslint-config-node
```

```js
// .eslintrc.js

module.exports = {
  extends: ['@commercetools-backend/eslint-config-node'],
};
```

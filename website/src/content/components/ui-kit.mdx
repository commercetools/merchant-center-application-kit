---
title: UI-Kit
---

To develop Custom Applications, commercetools provides a [UI component library](https://uikit.commercetools.com) that implements commercetools **Design System**.

The library provides all the necessary building blocks to build and compose the final user interface, for example buttons, input fields, table, spacings components, etc.

<Info>

Please refer to the [UI-Kit website](https://uikit.commercetools.com) to explore all the available components and design tokens. The website is built with Storybook and allows to interact with the components using **Knobs**.

</Info>

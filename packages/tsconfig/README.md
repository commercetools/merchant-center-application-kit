TSConfig file used by a Custom Application in TypeScript.

## Install

```bash
$ npm install --save @commercetools-frontend/tsconfig
```

## Usage

Add to your `tsconfig.json`:

```json
{
  "extends": "@commercetools-frontend/tsconfig"
}
```

## Getting started

Install the dependencies (uses yarn workspaces):

```bash
$ yarn
```

Build the application bundles

```bash
$ yarn build

# or
$ yarn build:es:watch
```

Start the development server (example app defined in [playground](./playground) folder):

```bash
$ yarn start
```

To run the tests:

```bash
$ yarn test

# or
$ yarn test:watch
```

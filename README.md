## Getting started

Install the dependencies (uses yarn workspaces):

```bash
$ yarn
```

Start the development server (example app defined in [playground](./playground) folder):

```bash
$ yarn start
```

To run the tests:

```bash
$ yarn test

# or
$ yarn test:watch
```
